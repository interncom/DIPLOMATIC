API

PEEK
GET /sigs?lo=CHECKED_AT
- returns list of SIGS as well as NOW (timestamp)

PUSH
POST /ops sends a batch of ops
- how to authenticate? signs the payload in header?

FETCH
POST /fetch to fetch a batch of ops
- payload is list of SIGs?
- host returns OPS concatenated together with each other + error records

CHECK
HEAD /sigs?salt=SALT&to=SYNCED_AT
- returns checksum of salted SIGs recorded up to SYNCED_AT

AUDIT
HEAD /ops?salt=SALT&to=SYNCED_AT
- lik HEAD /sigs, but checksums the salted complete ops
