# Plain Status Demo

A simple HTML/JS demo for DIPLOMATIC that displays the latest status message from a connected host.

## Features

- Text inputs for host URL and seed
- Button to load and display the latest status message
- Uses the DIPLOMATIC web client to sync and retrieve messages

## Running

1. Install dependencies: `bun install`
2. Start the dev server: `bun run dev`
3. Open the displayed URL (usually http://localhost:5173/) in a browser
4. Enter a valid host URL and seed, then click "Load Status"

The demo will sync with the host and display the latest status message.
